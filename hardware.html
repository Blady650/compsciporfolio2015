The Basic Parts Of A Computer


Motherboard
The motherboard is a circuit board that looks like a flat metal plate. Here it allows power to every computer hardware like the CPU, monitor, and RAM. It also allows the hardware to communicate with each other. Every motherboard looks different, but they all have RAM slots, CPU slots, connectors, and inputs/outputs. It is similar to our Nervous system, where it sends signals to one part of the body to another.

CPU (Central Processing Unit)
CPU stands for Central Processing Unit, and its job is to run operating systems, run programs, and processes them (thus giving the name Processor). It looks like a small, flat, squared block. This is like the brain of the computer. Without it, the computer can't even run its OS. It's basically useless.

RAM (Random Access Memory)
Ram stands for Random Access Memory, and its job is to read data. For example if you open a program, it loads from the hard drive to the RAM so that is can read it much faster than the hard drive. The more GB it has, the more programs it can run without lag. If the computer has no RAM, it will still open programs but not fast and if there are multiple programs then it can crash. RAM looks like a long piece of computer chip that is installed onto the motherboard

Hard Drive
Hard Drive is a box that stores all of the data like your files and programs. The more GB it has, the more programs and files it can hold. Without a hard drive, the computer can't save files and programs. It is similar to a human's memory inside the brain.

Power Supply
The Power supply is what gives power to the motherboard, which gives power to every other hardware that is connected to the motherboard like lights, CPU, fans, etc. Without the power supply, the computer can't run.

Monitor
The monitor is a display of the computer. It physically shows you the operating system and everythign else. Without it, you have no idea what you are doing with the computer.

Keyboard
A keyboard is a piece of hardware that lets you put text in the computer. It has the ABCs, Numbers, Controls, and Symbols.

Mouse
A mouse is used to click programs, files, and other things on the computer screen. Without it, it's pretty difficult to navigate through your computer.
