An Introduction To HTML
HTML (HyperText Markup Language) is a type of language used for making websites. It was discovered by Tim Berners in 1990 and ever since then, we'd been using it for websites. HTML is used up of "tags" (ex. < head >< /head >) that organize the text you put! How convenient! To add more style to your webpage, we can add a .css to the html!



CSS (Cascading Style Sheet)


Css is also a language similar to html, but instead of making a website it organizes and styles your website better than if you style it inside html. It was invented by HÃ¥kon Wium Lie who also invented Opera, a browser.

With Css, we can be more accurate, more options, and gives the website a great look!

These are the reasons why it's better to add an .css to your website

  • Instead of putting style in html everytime, you can use css to make it so that you only have to put it one time. It makes it easier for us.
  • Css can make a text of line this style and another line way different. Add more variety to your website!
  • More options than html so that your website can be neat and professional

When using CSS, it should look like this:

h1 {

background-color: #33D6FF;

text-align: center;

}

If you place this text in CSS, whatever you put in "h1" will have color and have the text in the center.


Tags

Tags are used in Html in order to show text in websites, add images, make tables, and many more!

It is very simple to write a tag. Just don't forget to have < and > in between the tag (ex. <"html">). To close the tag, add a / (slash) in the beginning (ex. <"/html">).

Here are examples of tags

<"head">

This tag puts whatever you put between <"head"> and <"/head"> on top of your page. For example you can put the title of the website between head so that the title can be on the top!

<"br/">

This tag makes space between lines of text. Similar to pressing enter on a text document. It is used if the text are very close together and you want them separated.

<"img src=">

This tag makes it so that you can add an image to your website. Just place the link to the image after the equals sign and it will appear (Don't forget to close it!)

Example in HTML: <'img src="https://warosu.org/data/ck/img/0052/12/1392872133431.jpg" />

In The Website:

<"a href=">

This tag, similar to the above tag, makes it so that your website can have a link. Again, add the web address after the equals sign and it will show!

Example in HTML: <'a href="http://Google.com"> [Whatever you want to name it]<'/a'>
